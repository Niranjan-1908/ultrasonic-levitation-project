# Ultrasonic Levitation Project

## Overview
This project demonstrates ultrasonic levitation using Arduino and ultrasonic transducers. High-frequency sound waves create standing waves capable of suspending lightweight particles in air.

## Objective
To design and implement a low-cost ultrasonic levitation system for contactless particle manipulation.

## Components Used
- Arduino Uno
- Ultrasonic Transducers (40 kHz)
- Motor Driver
- Power Supply
- Breadboard

## Working Principle
Two ultrasonic transducers generate acoustic standing waves. Pressure nodes formed between waves trap small particles causing levitation.
